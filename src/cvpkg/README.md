このパッケージは三須が使います。OpenCVを使って画像認識を行います。

やることメモ
・画像ファイルを見つけてくるように設定する。2/5
・ボールの認識の精度を上げる。2/5
・ダンボールを認識してもらう。そしたら精度を上げる。2/5

これができたら、次はカメラで実装する。
・カメラのデータを取り込めるようにする。2/6
カメラのデータを送って、逐次計算を行ってもらう。
カメラのデータといえど静止画を連続的に切り替えているにすぎないため、カメラのデータをそのまま読み込ませればうまくいくように思う。これが言葉は簡単だが実装に時間がかかりそう。計算はまあ間に合うでしょう。間に合わなかったらまた考えます。

カメラのデータを取り込めるようになったらほぼ勝ちです。あとは実践あるのみです。

・ボールとの距離を計算する。2/7
まあこれは直径を計算すればどうにかなるでしょう。
・ボールのグローバルな位置を特定する。2/7
個人的にLiDARと組み合わせてみたいと考えています。グローバル座標系としてボールの位置も割り出すことができればとても便利になると思いました。
