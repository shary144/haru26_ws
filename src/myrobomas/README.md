コントローラーの入力情報を使いたいとき
ros2 run joy<パッケージ名> joy_node<ノード名>
これで、topic[/joy]に、コントローラーの情報が流れる

topicの情報を見たいとき
あたらしいterminalを開きましょう
ros2 topic echo /joy<ノード名>
これで、/joyの状態を見られます。これでコントローラーの入力が見られる。


このコードを使って、ロボマスを動かす手順
sudo chmod 777 /dev/ttyACM0を入力して、権限を付与

ros2 launch robomas_controller robomas_bridge.launch.py
でノードを起動する

ロボマスに電圧を与えます（赤いスイッチを解除する）

緑色のLEDがチカチカしたらOK

次に、緑のスイッチがあるので、そちらを押すと、常時点灯する。押されている状態から押されていない状態に変わったら通電するらしいので、最初がONだったら、2回押す必要がある。

これで、ロボマスとの接続は完了しました。
後は、
ros2 run myrobomas my_robot_nodeを実行すると、私のコードが動きます。
だけど、
ros2 run joy joy_node
これも必要です。
これで、topic[/joy]に、コントローラーの情報が流れる



mode点灯なしの場合のボタンの配列
=== axes (float32) ===

[0] 左スティック X軸
    左: +1 / 右: -1

[1] 左スティック Y軸
    上: +1 / 下: -1

[2] 右スティック X軸
    左: +1 右: -1

[3] 右スティック Y軸
    上: +1 / 下: -1

[4] 十字キー 左 / 右
    左: -1 / 右: +1

[5] 十字キー 上 / 下
    上: +1 / 下: -1


=== buttons (int32) ===

[0]  X ボタン
[1]  A ボタン
[2]  B ボタン
[3]  Y ボタン
[4]  LB（左バンパー）
[5]  RB（右バンパー）
[6]  LT
[7]  RT
[8]  BACK
[9]  START
[10] 左スティック押し込み
[11] 右スティック押し込み
