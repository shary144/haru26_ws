mode点灯なしの場合のボタンの配列

=== axes (float32) ===

[0] 左スティック X軸
    左: +1 / 右: -1

[1] 左スティック Y軸
    上: +1 / 下: -1

[2] 右スティック X軸
    左: +1 右: -1

[3] 右スティック Y軸
    上: +1 / 下: -1

[4] 十字キー 左 / 右
    左: -1 / 右: +1

[5] 十字キー 上 / 下
    上: +1 / 下: -1


=== buttons (int32) ===

[0]  X ボタン
[1]  A ボタン
[2]  B ボタン
[3]  Y ボタン
[4]  LB（左バンパー）
[5]  RB（右バンパー）
[6]  LT
[7]  RT
[8]  BACK
[9]  START
[10]  左スティック押し込み
[11] 右スティック押し込み
